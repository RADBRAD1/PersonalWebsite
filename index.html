MERN stack Hotel Booking App (MongoDB, Express, Node.js, React)

In this project, I created a full-stack hotel booking application for residents in the mythical world of westeros. To create the static website elements for users, I used typescript with tailwind css. I created multiple API endpoints and routing to connect the backend and frontend elements using express and Node.
